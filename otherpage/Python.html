基本認識

- 資料型別
int - 整數 ex:1,2,3
str - 字串 ex:"我是個字串"
float - 浮點數 ex:0.01
bool - 布林值 ex:True
list - 列表 ex:[1,2,3]
tuple - 元組 ex:(1,2,3)
dict - 字典 ex:{'apple':3,'banana':4}
set - 集合 ex{1,2,3}
以上為常見的資料型別 細項之後會介紹

- 運算字符
() - 優先計算
** - 次方
* - 乘法
/ - 除法
// - 整除
% - 求餘數
+ - 加法
- - 減法
> - 大於
< - 小於
== - 相等
<= - 小於等於
>= - 大於等於
!= - 不等於
and - 雙方都需等於True才會輸出True
or - 只要有一方等於True就會輸出True
not - 相反
以上按照優先級排序

- 基本函式
print() - 列印出()內的東西在終端機
len() - 計算列表中有多少個東西
input() - 獲取用戶輸入的值
int()、float()、str() - 轉換資料型態
range(起始值,終止值,間隔值) - 創建一個整數數列 常搭配for迴圈使用


- 常用模塊
math - 數學模塊提供數學運算函式
random - 提供生成隨機數的函式
datatime - 時間相關函式
os - 訪問文件 文件交互操作
json - 處理json數據
requests - HTTP相關數據
tkinter - 做GUI的


For while 迴圈

- For迴圈
寫法為： 

for <變數> in range(起始值,終止值,間隔值):
                        
    print(<變數>)
                        
此代碼將會輸出range中的所有內容
                        
也可將 in 後面的 range() 改為列表或字串

- while迴圈
寫法為： 

while <條件>:
                        
    print('awa')
                        
當條件符合時此代碼會一直印出awa
